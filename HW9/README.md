# Week 9 遙測作業 - ARIA v6.0 驗證分析系統

## 專案概述

本專案為台灣大學遙測與空間資訊分析課程 Week 9 作業，實作 **ARIA v6.0 (Automated Remote Imaging Auditor)** 驗證分析系統，以台灣馬太安堰塞湖事件為案例，進行專業級災害影響評估與精度驗證。

## 核心功能

### 🛰️ 多光譜變異檢測
- **NDVI** (正規化植被指數)：檢測植被覆蓋變化
- **NDWI** (正規化水體指數)：偵測淹水範圍與水體變化
- **BSI** (裸土指數)：識別裸露土壤與土石流區域
- **強制雲遮罩**：使用 Sentinel-2 SCL 層避免「假水體」偵測誤差

### 📊 精度驗證與門檻優化
- **F1-score 門檻掃描**：測試 5+ 個 ΔNDVI 門檻值
- **混淆矩陣分析**：計算 OA、PA、UA、Kappa 等精度指標
- **地面實況驗證**：使用教師提供的 60 個驗證點進行量化評估

### 🗺️ 三區信心度分類
- **高信心區**：|ΔNDVI| > 1.5 × threshold (強烈變異訊號)
- **低信心區**：threshold < |ΔNDVI| ≤ 1.5 × threshold (邊界變異)
- **無偵測區**：|ΔNDVI| ≤ threshold (無顯著變化)

### 📋 專業災害報告
- **執行摘要**：關鍵發現與行動建議
- **變異分析**：多光譜指標結果與空間分布
- **精度評估**：驗證結果與信心度分析
- **作業建議**：針對疏散規劃與監測團隊的具體建議

## 技術架構

### 資料來源
- **Sentinel-2 L2A**：10m 解析度多光譜衛星影像
- **Microsoft Planetary Computer**：STAC API 影像搜尋與下載
- **地面實況**：教師提供的驗證點 GeoJSON

### 分析流程
1. **環境設定**：載入套件、設定 BBOX、讀取環境變數
2. **影像搜尋**：取得 Pre/Mid/Post 三期 Sentinel-2 影像
3. **雲遮罩處理**：使用 SCL 層建立有效像元遮罩
4. **指標計算**：計算 NDVI、NDWI、BSI 及變異圖層
5. **門檻優化**：透過 F1-score 找出最佳分類門檻
6. **精度評估**：建立混淆矩陣與精度指標
7. **信心度分區**：建立三區信心度地圖
8. **報告整合**：生成專業災害評估報告

## 主要特色

✅ **驗證導向**：從探索性分析轉向量化驗證評估  
✅ **雲遮罩強制要求**：避免遙測分析中的常見陷阱  
✅ **多指標整合**：NDVI、NDWI、BSI 三種指標交叉驗證  
✅ **信心度分級**：為決策者提供不同風險等級的行動指引  
✅ **可重現性**：透過 .env 檔案確保分析結果可重現  

## 使用方式

1. 設定 `.env` 檔案中的 Sentinel-2 Item IDs 或目標日期
2. 依序執行 8 個分析 cells
3. 檢查各項輸出圖表與精度指標
4. 確認生成的災害評估報告

## 輸出成果

- **2×2 變異檢測圖**：ΔNDVI、ΔNDWI、ΔBSI 空間分布
- **門檻優化圖表**：F1-score、PA、UA 隨門檻變化趨勢
- **混淆矩陣**：2×2 分類精度評估
- **信心度地圖**：三區分類的空間分布與面積統計
- **專業報告**：整合分析的 Markdown 格式災害評估
