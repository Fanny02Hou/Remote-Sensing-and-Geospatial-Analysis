
## Task 1: K-means Unsupervised Classification — Discussion

本研究首先使用 Sentinel-2 的六個反射率波段（B02、B03、B04、B08、B11、B12）進行 K-means 非監督式分類，並設定 K = 5，以探索秀林／太魯閣研究區中的主要光譜群集。由於 K-means 並不使用人工標記的訓練資料，因此其輸出結果僅代表光譜相似性所形成的 cluster，而非模型自動理解的真實土地覆蓋類別。

從平均光譜特徵來看，Cluster 1 的各波段反射率皆偏低，且 NIR 與 SWIR 反射率也低，因此較容易解釋為水體。Cluster 0 具有明顯高 NIR 與低 Red 的特徵，NDVI 較高，符合森林或高密度植被的光譜表現。Cluster 3 也呈現植被特徵，但 NIR 反射率低於 Cluster 0，因此可初步解釋為農地、草地或低密度植被。Cluster 4 的可見光、NIR 與 SWIR 反射率皆偏中高，較符合裸地或崩塌地。Cluster 2 則在所有波段中反射率皆最高，可能對應亮色建物、都市地表、裸露地或高反射人工地物。

較容易解釋的類別包括水體與森林，因為水體通常呈現低 NIR、低 SWIR，而森林則呈現高 NIR、低 Red。較難解釋的是建物、裸地與崩塌地，因為這些地物在可見光與 SWIR 波段中可能具有相似的高反射特徵，容易被 K-means 分到相近或混合的 cluster。此外，山區地形陰影、雲影與水體都可能呈現低反射率，因此也會造成非監督式分類的混淆。

整體而言，K-means 可作為影像分類前的探索性分析工具，協助理解研究區內主要光譜群集，但其 cluster 編號沒有固定意義，也無法保證每個 cluster 都對應單一真實地物類別。因此，後續仍需要透過 Random Forest 監督式分類，利用人工 ROI 訓練樣本建立具有明確地物標籤的土地覆蓋分類圖。
